## Instalacija

Za pokretanje programa potrebno je instalirati node.js [nodejs](https://pip.pypa.io/en/stable/).

Nakon toga je potrebno instalirati module iz package.json datoteke.

```bash
cd src
npm install
```

## Pokretanje programa

```python
node index.js
```
